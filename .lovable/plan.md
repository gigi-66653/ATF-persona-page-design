
## Persona 卡片重新设计

### 设计风格
- **毛玻璃质感**：卡片背景使用半透明模糊效果（backdrop-blur），配合柔和蓝色主色调
- **柔和配色**：以浅蓝色渐变为主，搭配白色/半透明元素
- **大圆角 + 柔和阴影**：营造高级感

### 单张卡片信息层级（从上到下）
1. **Persona Name**（大字号，加粗，视觉焦点）
2. **Persona 描述**（次级焦点，适中字号，柔和灰色）
3. **核心维度标签**（3个标签，pill 样式，半透明背景）
4. **品牌关系**（自定义字段区域，如 Loyalty 状态 + 关系描述，用小图标+文字组合）
5. **Posts 数量**（底部，最小层级信息）

### 交互
- Hover 时卡片微微上浮 + 阴影加深
- 点击可展开详情区域（展开时显示更多维度信息或详细分析）
- 展开/收起有平滑动画

### 实现
- 创建 `PersonaCard` 组件，支持 props 传入所有字段
- 用 demo 数据展示 2-3 张卡片的网格布局，呈现整体效果
- 使用 Tailwind 实现毛玻璃、渐变、动画效果
